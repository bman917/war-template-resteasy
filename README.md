## Project Description

Example WAR Build using REST Easy

## Build

```bash
./gradlew war
```

## Run

```bash
./run.sh
```

## Access

```
http://localhost:8080/war-template-resteasy/rest/greet/json
http://localhost:8080/war-template-resteasy/rest/greet/xml
```
